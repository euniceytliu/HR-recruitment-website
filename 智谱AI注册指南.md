## 📋 注册步骤（3分钟）

### 步骤1：访问官网

👉 打开浏览器，访问：**https://open.bigmodel.cn/**

### 步骤2：注册账号

1. 点击右上角「**登录/注册**」按钮
2. 选择注册方式：
   - **手机号注册**（推荐）
     - 输入手机号
     - 获取验证码
     - 设置密码
   - **微信登录**（最快）
     - 扫码即可

### 步骤3：获取 API Key

1. 注册成功后，点击右上角**头像**
2. 选择「**API Keys**」
3. 点击「**创建 API Key**」按钮
4. 给 Key 起个名字（比如：`简历分析器`）
5. 点击「**确定**」
6. **立即复制** API Key（格式类似：`xxx.xxxxxxxxxxxxxxxxxx`）

⚠️ **重要**：API Key 只会显示一次，一定要立即保存！
