# 🌟 腾讯HR职业成长伙伴网站 - 使用指南

## 📋 网站定位

**核心价值：** 成为学生的"职业成长伙伴"，通过提供独家洞察、个性化诊断和沉浸式体验，帮助学生在求职腾讯HR岗位的旅程中，清晰地认识自我、了解岗位，并规划职业路径。

**目标用户：** 对腾讯HR岗位感兴趣的在校学生

**核心差异化：** 不只是发布JD，而是提供JD背后的"说明书"和真实工作体验

---

## 🗂️ 网站架构

### 1. **首页 (index.html)**
- **定位：** 网站入口，建立情感连接
- **核心内容：**
  - Hero Banner：职业成长伙伴的价值主张
  - 三大核心功能卡片：HR的一天 / HR知识库 / 热招岗位
  - 个人介绍：建立信任感
  - 为什么选择HR职业

### 2. **一名腾讯HR的一天 (hr-day.html)**
- **定位：** 打破刻板印象，展示真实工作场景
- **核心功能：**
  - 互动时间轴：从早9点到晚7点的7个关键时刻
  - 真实场景：简历筛选、业务沟通、面试组织、数据复盘等
  - 能力标签：每个场景标注所需的核心能力
  - 数据可视化：展示工作成果（如：23份简历→15份通过）
  - 情感共鸣：候选人反馈、工作温度

**设计亮点：**
- 左右交替的时间轴布局（桌面端）
- 悬停动画效果
- 渐变色彩搭配
- 移动端响应式设计

### 3. **HR知识库 (hr-knowledge.html)**
- **定位：** JD背后的"说明书"，构建知识体系
- **核心功能：**
  - **JD解读器：** 对比"JD怎么写"vs"实际需要什么"
  - **三大岗位深度解读：**
    1. 招聘专员/HR BP
    2. 培训与发展
    3. HRBP（业务伙伴）
  - **每个岗位包含：**
    - 核心能力要求（5-6项）
    - 腾讯真实案例
    - 推荐学习资源（书籍、课程、认证）

**设计亮点：**
- 卡片式布局
- 渐变色区分不同岗位
- 资源链接带图标
- 可扩展的案例展示

### 4. **热招岗位 (jobs.html)**
- **定位：** 实际招聘入口
- **核心功能：**
  - 从Supabase数据库动态加载岗位
  - 热招标记
  - 岗位要求展示
  - 一键申请链接

### 5. **岗位管理后台 (jobs-admin-simple.html)**
- **定位：** HR专用管理工具
- **核心功能：**
  - 添加新岗位
  - 删除岗位
  - 查看所有岗位
  - 不依赖CDN，使用原生fetch API

## 🎨 设计特色

### 视觉风格
- **配色方案：**
  - 主色调：紫色渐变 `#667eea → #764ba2`（专业、创新）
  - 辅助色：蓝色渐变 `#4facfe → #00f2fe`（信任、活力）
  - 强调色：粉红渐变 `#f093fb → #f5576c`（温度、热情）

- **设计元素：**
  - 圆角卡片（15px border-radius）
  - 柔和阴影（box-shadow）
  - 悬停动画（hover效果）
  - 渐变背景

### 交互体验
- **响应式设计：** 完美适配桌面端、平板、手机
- **流畅动画：** fadeIn、translateY动画
- **视觉层次：** 清晰的信息架构
- **情感化设计：** Emoji、真实案例、温暖文案

## 🚀 使用流程

### 给学生的使用路径：
1. **首页** → 了解网站定位和价值
2. **HR的一天** → 沉浸式体验真实工作
3. **HR知识库** → 深度了解岗位要求和学习资源
4. **热招岗位** → 查看并申请心仪岗位

### 给HR的管理流程：
1. 打开 `jobs-admin-simple.html`
2. 填写岗位信息（名称、部门、地点、要求等）
3. 勾选"热招岗位"和"已发布"
4. 点击"保存岗位"
5. 岗位立即出现在 `jobs.html` 页面

## 🔧 技术说明

### 技术栈
- **前端：** 原生HTML + CSS + JavaScript
- **数据库：** Supabase (PostgreSQL)
- **部署：** 静态网站托管（GitHub Pages / Netlify）

### 文件结构
```
my first website/
├── index.html                  # 首页
├── hr-day.html                 # HR的一天
├── hr-knowledge.html           # HR知识库
├── jobs.html                   # 热招岗位展示
├── jobs-admin-simple.html      # 岗位管理后台
├── style.css                   # 全局样式
├── profile.jpg                 # 个人照片
└── 腾讯HR职业成长伙伴网站-使用指南.md
```

### 数据库配置
- **URL:** `https://gevvmjwjmpjhwczfuiru.supabase.co`
- **表名:** `jobs`
- **字段：**
  - `id`: 自增主键
  - `title`: 岗位名称
  - `department`: 部门
  - `location`: 工作地点
  - `requirements`: 岗位要求（数组）
  - `is_hot`: 是否热招
  - `apply_url`: 申请链接
  - `published`: 是否发布
  - `created_at`: 创建时间
