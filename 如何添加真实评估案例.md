# 📋 如何添加真实HR评估案例

## 🎯 为什么要添加真实案例？

AI 学习你的**真实评估案例**后，可以：
- ✅ 更准确地模仿你的评估风格和标准
- ✅ 给出更符合腾讯HR真实要求的建议
- ✅ 评分标准与你的实际评估一致
- ✅ 识别你最看重的能力维度

---

## 📁 案例存储位置

所有真实案例存储在：
```
hr-real-cases.js
```

---

## ✍️ 如何添加你的真实案例

### **步骤 1：打开案例文件**

用代码编辑器打开 `hr-real-cases.js`

---

### **步骤 2：选择岗位类型**

根据你要添加的案例类型，找到对应的部分：

```javascript
communication: [  // 沟通型岗位案例
    // 在这里添加沟通型案例
],

analytical: [  // 分析型岗位案例
    // 在这里添加分析型案例
],

creative: [  // 创意型岗位案例
    // 在这里添加创意型案例
]
```

---

### **步骤 3：填写案例模板**

复制以下模板，填入你的真实评估数据：

```javascript

    score: 85,  // 你给这份简历的真实评分（0-100）

    // 候选人简历概况（简要描述）

        school: '211高校',  // 学校层次
        major: '人力资源管理',  // 专业
        gpa: 3.7,  // GPA
        internship: 'XX公司HR实习',  // 实习经历
        activities: '学生会副主席'  // 校园活动
    },
    
    // 你认为的优势（你在评估时写的优点）
    strengths: [
        '【人际敏感度】具体的优势描述+量化证据',
        '【沟通影响力】具体的优势描述+成果数据',
        '【服务意识】具体的优势描述+案例',
        '【团队协作】具体的优势描述+亮点'

    // 你认为的不足（你在评估时写的缺点）
    improvements: [
        '【数据分析】具体的不足+对标腾讯要求的差距',
        '【专业深度】具体的不足+影响',
        '【创新思维】具体的不足+建议方向',
        '【业务理解】具体的不足+改进点'
    ],
    
    // 你给候选人的建议（你的真实建议）
    suggestions: [
        '【1-3个月】短期行动计划',
        '【3-6个月】中期能力提升建议',
        '【6-12个月】长期实战项目建议',
        '【职业规划】腾讯HR职业路径建议',
        '【简历优化】如何更好呈现经历'
    ],
    
    // 你的评估总结（可选，但很有用）
    evaluatorComments: '你对这个候选人的整体评价和建议'
```

## 🎯 添加案例的最佳实践

### **1. 案例多样性**

确保你的案例库包含：
- ✅ **高分案例**（85-100分）：优秀候选人的标准
- ✅ **中等案例**（70-84分）：合格但有提升空间
- ✅ **低分案例**（60-69分）：不太匹配的候选人
- ✅ **不同背景**：985/211/普通本科，不同专业，不同实习经历

### **2. 评价要具体**

❌ **不好的例子**：
```
"沟通能力强"
"实习经历丰富"
```

✅ **好的例子**：
```
"【人际敏感度】字节跳动HR实习，独立负责15场宣讲，候选人满意度4.8/5.0，证明沟通和服务能力"
"【数据分析】用Python分析10万+简历数据，优化筛选流程，效率提升40%"
```

### **3. 对标腾讯要求**

所有不足都要**对标腾讯的真实要求**：

```
"【数据分析】简历中缺少数据驱动的案例，而腾讯HR需要用数据支持决策，建议补充..."
```

### **4. 建议要可执行**

不要只说"提升XX能力"，要给出**具体的行动计划**：

```
"【1-3个月】学习Excel数据分析，完成《HR数据分析入门》课程"
"【3-6个月】参加1-2个HR相关竞赛，或主导1个中型校园活动"
```

## 🔄 更新提示词（让AI学习案例）

添加完案例后，需要更新后端 API，让 AI 能读取这些案例：

### **方法 1：手动更新（简单）**

1. 打开 `api/analyze-resume.js`
2. 找到 `buildPrompt` 函数（第 122 行左右）
3. 在 `# 真实评估案例学习` 部分，把你的新案例**复制粘贴**进去

### **方法 2：自动引用（高级）**

修改 `api/analyze-resume.js`，在文件顶部添加：

```javascript
const realCases = require('../hr-real-cases.js');
```

然后在 `buildPrompt` 函数中动态插入案例。

## 💡 建议添加多少案例？

- **最少**：每个岗位类型 2-3 个案例（高分、中分、低分各1个）
- **推荐**：每个岗位类型 5-10 个案例
- **最佳**：每个岗位类型 10-20 个案例，覆盖各种背景

案例越多，AI 学习效果越好！
