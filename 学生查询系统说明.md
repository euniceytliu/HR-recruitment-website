# 📋 学生案例评估结果查询系统

## 🎯 功能概述

学生可以通过 **手机号 + 邮箱** 验证身份，查看自己提交的案例评估结果和详细反馈。

---

## 🔗 访问链接

### 本地开发环境
```
http://localhost:8888/case-result.html
```

### 生产环境（部署后）
```
https://你的域名/case-result.html
例如：https://tencent-hr-website.vercel.app/case-result.html
```

---

## 📱 学生使用流程

### 第1步：提交案例
1. 学生在 `application.html` 页面填写案例信息
2. 填写时记录：**手机号** 和 **邮箱**
3. 点击提交

### 第2步：等待评审
- 通常 3-5 个工作日
- HR在后台进行评估（手动或自动）

### 第3步：查询结果
1. 访问 `case-result.html` 查询页面
2. 输入提交时填写的 **手机号** 和 **邮箱**
3. 点击"查询结果"按钮

## 🎨 界面设计特点

### 1. **渐变背景**
- 紫色渐变（#667eea → #764ba2）
- 与腾讯品牌色系一致

### 2. **圆形分数展示**
- 动画圆环进度条
- 根据分数自动计算百分比
- 视觉冲击力强

### 3. **卡片式布局**
- 白色卡片 + 阴影
- 圆角设计，现代感强
- 移动端友好

### 4. **色彩语义化**
- 🟢 绿色 = 已评审 / 优势
- 🟡 橙色 = 待审核 / 改进
- 🔵 蓝色 = 建议 / 提示
- 🔴 红色 = 未通过 / 警告

### 5. **图标丰富**
- ✓ ✨ 📊 💼 🎯 等emoji图标
- 增强视觉表达
- 易于快速识别

## 🔒 安全机制

### 1. 双重验证
- 必须同时输入 **手机号** 和 **邮箱**
- 两者都匹配才能查询
- 防止他人恶意查询

### 2. 数据隐私
- 只显示自己提交的案例
- 不显示其他学生的信息
- 敏感信息脱敏处理

### 3. 最近记录
- 只返回最新的一条匹配记录
- 避免历史记录混淆

### 方式1：在提交成功页面提示（推荐）

```javascript

    <br>

```

```
感谢您的参与！

腾讯HR团队
```

### 方式3：在网站首页添加入口

在 `index.html` 添加查询入口：


## ❓ 常见问题

### Q1: 学生忘记手机号或邮箱怎么办？
**A**: 目前无法找回。建议：
- 在提交成功页面提示学生记录信息
- 可以联系HR团队人工查询

### Q2: 一个手机号提交多次，显示哪个？
**A**: 显示**最新的**一条记录。

### Q3: 可以修改查询验证方式吗？
**A**: 可以。当前是手机号+邮箱，也可以改为：
- 提交编号 + 手机号
- 身份证后4位 + 手机号
- 邮箱验证码

### Q4: 学生可以重新提交吗？
**A**: 可以。系统会保存多条记录，查询时显示最新的。
