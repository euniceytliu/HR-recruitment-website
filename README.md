# Eunice的个人博客

一个集成Supabase数据库的现代化个人博客网站。

## ✨ 功能特性

- 📝 动态文章管理（基于Supabase数据库）
- 🎨 响应式设计，支持各种设备
- 🚀 文章列表和详情页面
- 📊 实时从数据库加载内容
- 🔒 RLS行级安全策略保护

## 🛠️ 技术栈

- **前端**: HTML, CSS, JavaScript
- **数据库**: Supabase (PostgreSQL)
- **部署**: Vercel / Netlify / GitHub Pages

## 📦 文章内容

- 职业发展相关
- 人才招聘洞察
- AI时代HR转型
- 个人成长心得

## 🚀 本地运行

```bash
# 启动本地服务器
python3 -m http.server 8080

# 访问
http://localhost:8080/blog.html
```

或使用快捷脚本：
```bash
./start-server.sh
```

## 📝 添加新文章

在Supabase控制台的Table Editor中，向`articles`表插入新记录即可。
